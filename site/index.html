Operating model

This site is built for fast technical review: verify inventory, inspect evidence, and validate commands without guesswork.

  • Evidence is tied to repository paths and command-level reproducibility.
  • Counts match PROOF_PACK/VERIFIED_COUNTS.md and update via scripted generation.
  • Each major project includes direct evidence lanes for screenshots, writeups, and source references.

    Run verify scripts and match counts to the proof source.

    Review detections, playbooks, and migration artifacts by folder.
