How Verification Works

Three scripts, one truth

Every number on this site traces back to a single source of truth file generated by scripts that run against the actual repo artifacts. CI validates counts on every push and fails if anything drifts.

verify-counts.ps1
Counts detection rules and IR playbooks by walking the repo directories. Output matches PROOF_PACK/VERIFIED_COUNTS.md exactly. Run it locally — fast, no extra setup needed.
scripts/verify/verify-counts.ps1PowerShell
drift_scan.py
Compares hardcoded claim numbers in HTML against the verified counts source of truth. Fails CI if any public claim drifts. Skips lines with data-verified attributes.
scripts/drift_scan.pyPythonCI gate

Why Cloning Is Proof

Public + versioned
The repo is public. The commit history shows when each rule was added. The CI log shows every count validation run. Nothing is hidden or adjustable after the fact.

Published Outputs

What CI generates and publishes

CI doesn't just check counts — it generates the artifacts that back up every public claim. These outputs are committed to the repo on each validated run.

PROOF_PACK/VERIFIED_COUNTS.md
Markdown source of truth. Lists exact counts with source file references and line numbers. Auto-generated by generate-verified-counts.ps1. This is what everything else traces back to.
Auto-generatedSource of truth
site/assets/verified-counts.json
JSON artifact consumed by the site at runtime to hydrate count displays. Generated from VERIFIED_COUNTS.md. Allows real-time count display without JS being required for SEO crawlers.

All Wazuh rule XML files concatenated into a single deployable bundle. Built by build-wazuh-bundle.ps1. CI asserts the file exists after the build step. Deploy-ready for a Wazuh Manager instance.

Why crawlers used to show "0" (and why they don't now)

The old build relied on client-side JS counters. If JS didn't run (bots, locked-down browsers), the page rendered zeros. This site now renders numbers as plain HTML and only uses JS for optional UX (modals, copy buttons, live count hydration).
