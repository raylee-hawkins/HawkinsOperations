# Proof

Canonical public proof lane for HawkinsOps and SignalFoundry.

Use this directory as the reviewer-facing evidence entrypoint.

## Primary proof surfaces

- `verified-counts.md` - public verified inventory snapshot
- `autosoc/latest/` - latest public SignalFoundry proof artifacts generated by the AutoSOC engine
- `wazuh/honeypot/` - latest public honeypot proof artifacts

## Routing rule

`proof/` is the canonical public proof lane.

`PROOF_PACK/` remains available for legacy support, internal packaging, and process history, but it should not be used as the primary reviewer entrypoint in README, START_HERE, or site-first navigation.
