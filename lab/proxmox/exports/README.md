# Proxmox Shared Exports

Use this folder for shared Proxmox text exports only.

Allowed:
- Redacted command output files (`.txt`, `.md`) for reproducible evidence.

Not allowed:
- VM backups or disk images (`.vma`, `.qcow2`, `.zst`, `.iso`, `.tar.gz`).
- Secrets, tokens, keys, credentials, or internal IPs.

Use `[REDACTED_INTERNAL]` for sensitive internal details.
