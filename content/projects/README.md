# Projects

Portfolio project entrypoints live here.

Canonical public project lanes:

- `autosoc/` flagship AutoSOC supporting project content and curated mirrors
- `lab/` public lab projects and case studies

Internal, legacy, and reference-heavy project material should not become reviewer entrypoints.
If a folder here exists for archive, migration, repo-history, or reference purposes, keep it out of the site navigation and out of recruiter-first paths.

Canonical AutoSOC generated metrics do not live in `projects/`.
Use `content/metrics/autosoc_latest.json` for the public metrics source of truth.
