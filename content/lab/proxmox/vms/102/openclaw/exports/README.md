# OpenClo VM Exports

Store redacted text exports here.

Required format:
- Redacted `qm config 102` output as text (`.txt` or `.md`).

Never store:
- ISO files, disk images, or backups (`.iso`, `.qcow2`, `.vma*`, `.zst`, `.tar.gz`).
- Secrets, tokens, keys, credentials, or internal IPs.

Use `[REDACTED_INTERNAL]` for sensitive internal details.
